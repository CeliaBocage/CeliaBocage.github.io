Projets Techniques

Sélection de projets techniques réalisés à l'EPITA et lors de mon semestre en Irlande. Chaque projet m'a permis de monter en compétences sur des technologies variées, de travailler en équipe et de mener un développement de bout en bout.

Résolution de mots cachés par OCR (équipe de 4)

EPITA 3e semestre
  • Développement d'un logiciel capable de détecter et résoudre des grilles de mots cachés à partir d'une image, grâce à la reconnaissance optique de caractères (OCR).
  • Implémentation d'un réseau de neurones en C atteignant 99,9 % de précision sur la reconnaissance de l'alphabet français.
  • Création d'une base de données d'entraînement et gestion du pipeline complet : prétraitement d'image, segmentation, reconnaissance et résolution.
  • Développement de l'interface graphique avec GTK et Glade pour permettre à l'utilisateur de charger une image et visualiser le résultat.
C Machine Learning Analyse d'Image GTK

Puzzle/Escape Game 3D - Godot (équipe de 5)

EPITA 2e semestre
  • Escape game multijoueur dans un manoir : résolution d'énigmes en coopération tout en échappant à un fantôme doté d'un système de pathfinding (possibilité de se cacher pour le faire repartir).
  • Modélisation 3D complète de l'environnement avec Blender (manoir, décors, éclairage).
  • Développement d'un système de configuration des contrôles en C#, avec sauvegarde/chargement dynamique via JSON.
  • Intégration de l'ensemble dans le moteur Godot : gameplay, physique, IA ennemie et interface utilisateur.
C# Godot Blender JSON

Logiciel de recettes de cuisine

ATU Sligo, Irlande Semestre à l'étranger
  • Application desktop de gestion de recettes : ajout, suppression, recherche et filtrage par ingrédient.
  • Fonctionnalité de constraint programming : suggestion de recettes réalisables avec les ingrédients disponibles ("recettes à partir du frigo").
  • Calcul automatique des portions en fonction du nombre de convives.
  • Base de données MariaDB, requêtes SQL/LINQ, interface multi-pages en XAML.
C# SQL XAML LINQ
